Actividad de Aprendizaje No. 1

Página Web Completa con Diseño Responsive

Desarrollo de una página web completa utilizando HTML5 semántico y CSS3 con enfoque en diseño responsive que se adapte a diferentes dispositivos y resoluciones.

Ver Descripción Ver Implementación

Descripción de la Actividad

Objetivo General

Crear una página web completa con diseño responsive que demuestre el dominio de HTML5 semántico, CSS3 y principios de diseño web adaptativo.

Requisitos Específicos

  • Estructura HTML5 semántica: Uso adecuado de etiquetas como <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>
  • Diseño responsive: La página debe verse correctamente en dispositivos de diferente resolución (escritorio, tabletas, móviles)
  • Organización del contenido: Implementación de cabecera, menú de navegación, contenido principal y pie de página
  • Estilos CSS personalizados: Uso de hojas de estilo para dar apariencia atractiva a todos los elementos
  • Contenido organizado: Sección de noticias y barra lateral con contenido relacionado

Tecnologías Utilizadas

HTML5 CSS3 Responsive Design JavaScript Básico

Ejemplo de Media Query


/* Media query para tablets */
@media (max-width: 768px) {
    .contenido-principal-actividad {
        flex-direction: column;
    }
    
    .seccion-noticias {
        padding-right: 0;
        margin-bottom: 40px;
    }
    
    .menu-principal-actividad nav ul {
        flex-direction: column;
    }
    
    .nav-list {
        display: none;
        flex-direction: column;
    }
    
    .nav-list.active {
        display: flex;
    }
}
