ACTIVIDADES IMPLEMENTADAS
-------------------------
1. ACTIVIDAD 1 - Página Web Responsive
   • Página web completa con estructura semántica HTML5
   • Diseño responsive para diferentes dispositivos
   • Cabecera, menú de navegación, contenido principal y pie de página
   • Sección de noticias con grid layout
   • Barra lateral con promociones
   • Uso de media queries y diseño mobile-first

2. ACTIVIDAD 2 - Formulario de Registro Básico
   • Formulario de registro responsive
   • Validación básica con JavaScript
   • Campos: nombres, apellidos, fecha nacimiento, país, email, contraseña
   • Validación de campos obligatorios
   • Mensajes de error básicos
   • Botón para limpiar formulario

3. ACTIVIDAD 3 - Formulario Avanzado con Validación
   • Formulario de registro completo y responsive
   • Validación avanzada con JavaScript
   • Mensajes de error específicos debajo de cada campo
   • Validación en tiempo real
   • Confirmación de email y contraseña
   • Validación de edad mínima (18 años)
   • Términos y condiciones
   • Mensajes de éxito/error generales según especificaciones

CARACTERÍSTICAS COMUNES
-----------------------
• Diseño responsive para todos los dispositivos
• Menú de navegación con efecto hamburguesa en móviles

TECNOLOGÍAS UTILIZADAS

INSTRUCCIONES DE USO
--------------------
1. Abrir el archivo index.html en cualquier navegador moderno
2. Navegar entre las actividades usando el menú principal
3. Probar cada actividad individualmente:
   • Actividad 1: Ver página web responsive
   • Actividad 2: Probar formulario básico
   • Actividad 3: Probar formulario avanzado con validación completa
4. Verificar diseño responsive cambiando el tamaño de la ventana
5. Probar validaciones con datos correctos e incorrectos

VALIDACIONES IMPLEMENTADAS (ACTIVIDAD 3)
-----------------------------------------
• Nombres y Apellidos: Campo obligatorio, mínimo 2 caracteres
• Fecha de Nacimiento: Formato válido, no futura, mayor de 18 años
• País: Selección obligatoria
• Email: Formato válido (@ y dominio), confirmación de email
• Contraseña: Mínimo 8 caracteres, confirmación de contraseña
• Términos y condiciones: Aceptación obligatoria
• Validación en tiempo real para email y contraseña

RESPONSIVE DESIGN
-----------------
El sitio se adapta a:
• Escritorio (> 1024px): Layout completo con múltiples columnas
• Tabletas (768px - 1024px): Layout optimizado para pantallas medianas
• Móviles (< 768px): Una columna, menú hamburguesa, formularios apilados
